# Customer Purchasing Behavior Analysis

## Project Overview
This project analyzes customer purchasing behavior using Machine Learning techniques. It visualizes customer purchasing patterns and identifies customer segments using dimensionality reduction and clustering algorithms.

## Problem Statement
Visualize customer purchasing behavior in a 2D space using PCA or t-SNE to discover hidden patterns and segment customers using K-Means clustering.

## Algorithms Used
- Principal Component Analysis (PCA)
- t-Distributed Stochastic Neighbor Embedding (t-SNE)
- K-Means Clustering

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
